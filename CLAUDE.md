# Quantum-Link – Web Chat App Project

## Project Overview
Create a basic web chat application called **Quantum-Link** for a school project.

The goal is to build a simple, beginner-friendly, real-time chat app using JavaScript technologies.
The code should be easy to read, well structured, heavily commented, and simple to explain during a school presentation.

---

# Main Requirements

## 1. Technology Stack
Use JavaScript-based technologies only.

### Recommended Stack
- Frontend:
  - HTML
  - CSS
  - JavaScript

- Backend:
  - Node.js
  - Express.js

- Real-Time Messaging:
  - Socket.IO

- Database:
  - MongoDB (recommended)
  - OR simple JSON/local storage for beginner version

---

# 2. Code Style Requirements

The project must:
- Be beginner friendly
- Have clean folder structure
- Include comments throughout the code
- Use meaningful variable/function names
- Be easy to explain for a school project
- Avoid overly advanced programming patterns
- Keep functions short and readable

### Example
Good:
```js
function sendMessage() {
   // Sends the user's message to the server
}
```
